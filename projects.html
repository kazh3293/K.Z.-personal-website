This project manages characters and their items in a dungeon crawler game using a hash table with chaining. Each character has a priority queue for their items, organized by damage so the strongest item is always on top. The program also finds the easiest path through a dungeon using Dijkstra's algorithm, with item damage as edge weights.

This is a game requires two players If the player's Stamina, Strength, Wisdom, and Pride Points is less than 0, then these values will be 0 (non negative) First, there will be an introduction about the game. Then, the player need to enter their names and choose their characters Player need to enter relevant content as required. If the player fails to enter relevant content as prompted, the system requires the player to re-enter until relevant content is entered. Player can choose two different modes. Players need to go through numerous challenges to reach the finish line. In the end, the player with the highest Pride Points wins.
